# Automation & Utility Scripts

This folder is a placeholder for automation and helper scripts used during deployment, database configuration, or testing.

## Future Scope

Planned utilities for this folder include:
* **`deploy.sh` / `deploy.bat`:** Script commands to orchestrate docker-compose containers or execute runtime environment checks.
* **`db_migrate.py`:** Migration helper to handle transitioning the SQLite database configurations to PostgreSQL schemas.
* **`test_runner.py`:** Standard script execution command to trigger pre-commit integrations or unit verification files.
